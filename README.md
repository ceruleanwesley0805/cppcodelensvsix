# C++ CodeLens VSIX 安裝指南

此專案提供從 [cpp-codelens](https://github.com/WazWazMan/cpp-codelens) 打包的 VSIX 檔，方便在 Visual Studio Code 中啟用 C++ CodeLens 功能。

## 安裝步驟

1. **下載 VSIX 檔**
   - 從本專案的 `release` 或 `vsix` 資料夾取得 `.vsix` 檔案。

2. **在 VS Code 安裝**
   - 開啟 VS Code。
   - 前往 **Extensions** 視窗。
   - 點選右上角的 **...** → **Install from VSIX...**。
   - 選擇剛下載的 `.vsix` 檔案。

3. **啟用擴充功能**
   - 安裝完成後，重新啟動 VS Code。
   - 確認擴充功能已啟用。

## 功能簡介
- 顯示 C++ 函式的呼叫層級 (CodeLens)。
- 支援快速瀏覽與跳轉。
